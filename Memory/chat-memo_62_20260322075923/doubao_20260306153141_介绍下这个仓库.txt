点击「录入新素材」填写信息（标题、链接、项目等），提交后即可在列表页看到；
团队成员可通过「项目筛选」「分类筛选」快速找到对应素材，点击「查看原文」直接跳转到素材链接。
三、进阶优化建议（可选）
权限控制：如果需要限制访问，可在群晖「Web Station」中开启「基本认证」，设置用户名密码，仅团队成员可访问；

功能扩展：后续可增加 “素材标记（已选用 / 废弃）”“链接有效性检测”“批量导出” 等功能；
Docker 部署（可选）：如果不想用群晖自带的 PHP/MySQL，也可通过 Docker 部署（比如用 Docker Compose 运行 PHP+MySQL+Nginx），更灵活。
