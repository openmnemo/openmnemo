#### **@openmnemo/core** (`packages/core/`)
**Entry point:** `dist/index.js` | **Types:** `dist/index.d.ts`
**Build:** `tsup src/index.ts --format esm --dts`
**Dependencies:** `@openmnemo/types`, `fast-glob@3.3.3`, `sql.js@1.12.0`

**Source Structure:**
- `src/transcript/` - Parsing, import, discovery, database
  - `common.ts` - Constants (CLIENTS, TEXT_BLOCK_TYPES, SKIP_BLOCK_TYPES, TOOL_USE_TYPES, TOOL_RESULT_TYPES), utilities (slugify, sha256File, contentHash, timestamp handling, text extraction, deduplication)
  - `parse.ts` - `inferClient()`, `parseTranscript()`, `parseCodexTranscript()`, `parseClaudeTranscript()`, `parseGeminiTranscript()`
  - `discover.ts` - File discovery with glob patterns per client
    - `defaultGlobalTranscriptRoot()`, `defaultClientRoots()`, `discoverSourceFiles()`, `inferProjectSlug()`, `transcriptMatchesRepo()`, `projectSlugsMatch()`, `safeFileMtime()`
  - `import.ts` - Orchestration logic
    - `importTranscript()`, `transcriptHasContent()`, `writeCleanMarkdown()`, `preserveExistingImportTimestamp()`, `manifestSignature()`, `manifestChanged()`, `writeJson()`, `appendJsonl()`, `copyFile()`
  - `db.ts` - SQLite search index using sql.js
    - `upsertSearchIndex()`, `searchTranscripts()`
    - PK: client, project, session_id, raw_sha256
    - Data columns: title, started_at, imported_at, cwd, branch, raw_source_path, raw_upload_permission, global_raw_path, global_clean_path, repo_raw_path, repo_clean_path, repo_manifest_path, message_count, tool_event_count

- `src/recall/` - Session recall and sync
  - `recall.ts` - `recall()`, `syncCurrentProject()`, `findLatestSession()`, `findLatestFromJsonl()`, `cwdMatches()`, `formatRecallText()`
  - Exports `RecallResult` interface

- `src/storage/` - Storage abstraction
  - `adapter.ts` - `StorageAdapter` interface (async): `writeFile()`, `readFile()`, `exists()`, `mkdir()`, `copyFile()`, `appendFile()`
  - `local-adapter.ts` - `LocalAdapter` implementation using Node.js fs
  - `gitea-adapter.ts` - `GiteaAdapter` with options

- `src/utils/`

  - `exec.ts` - `execCommand()`, `git()` helper functions

   - Handler: `src/cmd-upgrade.ts` → `cmdUpgrade()`

   - Options: `--source` (required), `--root`, `--client` (auto|codex|claude|gemini), `--project-name`, `--global-root`, `--raw-upload-permission` (not-set|approved|denied), `--format` (text|json)
   - Handler: `src/cmd-import.ts` → `cmdImport()`

4. **`discover`** - Discover and import local AI transcripts
   - Options: `--root`, `--client` (all|codex|claude|gemini), `--scope` (current-project|all-projects), `--project-name`, `--global-root`, `--raw-upload-permission`, `--limit`, `--format`
   - Handler: `src/cmd-discover.ts` → `cmdDiscover()`

   - Options: `--root`, `--locale`, `--format`
   - Handler: `src/cmd-locale.ts` → `cmdLocale()`

6. **`recall`** - On-demand sync and latest session recall
   - Options: `--root`, `--project-name`, `--global-root`, `--activation-time`, `--format`
   - Handler: `src/cmd-recall.ts` → `cmdRecall()`

7. **`daemon`** - Heartbeat lifecycle management
   - Subcommands:

     - `uninstall` - Remove scheduled task
