### 5. **Import/Ingest Flow**
**File:** `C:/Users/ai/.codex/skills/memorytree-workflow/src/transcript/import.ts` (lines 21-108)

**Function:** `importTranscript(parsed, root, globalRoot, projectSlug, rawUploadPermission, mirrorToRepo)`

**Data flow:**
```
1. Parse JSONL → ParsedTranscript object
   - client, session_id, title, started_at, cwd, branch
   - messages: TranscriptMessage[]
   - tool_events: TranscriptToolEvent[]

2. Generate artifact stem from session_id + SHA256 hash (first 8 chars)
   - Format: `{sessionSlug}__{sha256[0:8]}`

3. Compute file paths (year/month partition):
   - Repo: Memory/06_transcripts/{raw|clean|manifests}/{client}/{year}/{month}/{stem}
   - Global: ~/.memorytree/transcripts/{raw|clean|manifests|index}/{client}/{project}/{year}/{month}/{stem}

4. Create directories & copy raw transcript file

5. Build ManifestEntry with:
   - message_count, tool_event_count, file paths, timestamps
   - cleaning_mode: 'deterministic-code'
   - repo_mirror_enabled: true/false

6. Write artifacts:
   - Clean markdown: writeCleanMarkdown(parsed, manifest, filePath)
   - Manifest JSON: writeJson(globalManifestPath, manifestRecord)
   - Append to event log: sessions.jsonl
   - Update search index: search.sqlite

7. Preserve original import_at timestamp if content hasn't changed (manifestSignature())

8. Return ManifestEntry
```

**Manifest lifecycle:**
- On re-import, `preserveExistingImportTimestamp()` keeps original `imported_at` if content signature matches
- Event log (`sessions.jsonl`) only appended if manifest differs from previous
- Search index upserted always

## Summary for Implementation

To add a new parser, follow this pattern:

1. **Create parser function** in `parse.ts`:
   ```typescript
   export function parseYourClientTranscript(filePath: string): ParsedTranscript {
     // 1. Load file (JSON/JSONL)
     // 2. Extract session metadata (id, title, timestamps, cwd, branch)
     // 3. Traverse structure to find messages with role + text
     // 4. Extract tool events (name + input/output)
     // 5. Deduplicate
     // 6. Return ParsedTranscript object
   }
   ```

2. **Update `inferClient()`** to detect your client from source path

3. **Update `parseTranscript()`** dispatch logic to call your parser

4. **Constants** (TEXT_BLOCK_TYPES, TOOL_USE_TYPES, etc.) in `common.ts` may need extension

5. **Text extraction** may need custom logic (create `extractYourClientText()` helper)

The rest (manifest generation, clean markdown writing, file paths, deduplication) is automatic.
